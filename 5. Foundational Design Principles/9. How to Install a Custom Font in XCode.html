1. Download the Custom Font

Google Fonts provides a large collection of free to use fonts. Xcode supports True Type Font (.ttf) and Open Type Font (.otf) font files which should be included in any font download from Google Fonts. If you are looking for font inspiration, you could try the WhatTheFont service to detect the fonts used by your favourite apps.



2. Import your Custom Font into XCode


Create a new group in your Project Navigator called Resources (this is optional but it will keep your project neat and tidy). Now simply drag and drop your font files into the Resources folder and ensure that "Copy items if Needed" and "Add to Targets" are selected.



It should now be possible to see your font files in your Project Navigator under the group Resources. Click on any one of them and ensure that Target Membership is selected in the File Inspector on the right hand side of Xcode.



3. Check that the Fonts Are Included in the Resource Bundle


Now go to the "Build Phases" of your project by clicking on your Xcode Project at the top of the Project Navigator. Look for a section called "Copy Bundle Resources" and expand it. You should be able to see your font files listed.



4. Include Your Fonts in your Info.plist


Now find your Info.plist in your Project Navigator and click it. Add a new key to your Info.plist called "Fonts provided by application". Now add the names of all the fonts under this key as an array. Use the name exactly as how it appears in your resources group. Ensure the "Target Membership" is selected for each font name you provide.



5. All Done. Check Everything Is Working


Go to Storyboard and drag out a UILabel. Try change the font to your custom font in Attributes Inspector. If you can find your custom font then everything is working correctly.



To use your custom font in code you will need to know its font name. This is not always obvious and might not match the file name. Run this code to get a list of all the font names in your app and try locate the font name of your custom font.
